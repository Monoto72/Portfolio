Project: Slider App

A revolutionary dating app designed to connect people through shared interests. Slider prioritizes meaningful connections by focusing on personality, not visuals, until after a match is made.

Check Out GitHub

Demo video showcasing the Slider App in action. Watch to see how it connects people based on shared interests.

Project Log

This project was developed over the course of a year by a team of 8 members, including developers, designers, and strategists. My role focused on backend development and UI layout implementation, working alongside another developer to ensure a smooth and user-friendly experience.

Tech Stack:

  • JavaScript: For app functionality
  • React Native: Cross-platform development
  • Firebase: Authentication, Firestore, and Cloud Storage

Highlights:

  • Developed the entire backend and implemented most UI layouts
  • Collaborated on design iterations for maximum usability
  • Integrated real-time updates, authentication, and media uploads
  • Ensured cross-platform compatibility for iOS and Android
  • Streamlined backend architecture using Firebase tools

Key Contributions:

  • Optimized database queries for better performance
  • Implemented secure user authentication workflows
  • Worked closely with the design team to align UX with project goals
  • Handled media upload and retrieval with Firebase Cloud Storage
